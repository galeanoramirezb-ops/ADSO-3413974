## Clasificación

| Severidad | Ejemplo | Respuesta esperada |
|-----------|---------|----------------------|
| Alta | El sistema no permite registrar ventas | Corrección inmediata; si no es posible, volver temporalmente al registro manual |
| Media | Error al generar un reporte | Corrección en el corto plazo |
| Baja | Detalle visual o de texto | Corrección en la siguiente actualización |

## Respuesta

1. Identificar y reproducir el problema.
2. Si afecta el registro de ventas, informar al personal del supermercado para usar el respaldo manual mientras se corrige.
3. Corregir, probar (ver [11-quality/testing-strategy.md](../11-quality/testing-strategy.md)) e instalar la corrección.
