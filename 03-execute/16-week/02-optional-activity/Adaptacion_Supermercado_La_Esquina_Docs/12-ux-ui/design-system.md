## Base

Se usa **Bootstrap** (definido en el SRS) como framework de UI, sin un sistema de diseño propio complejo, acorde al alcance del proyecto.

## Componentes principales

- Formularios de Bootstrap para login, alta de productos y registro de venta.
- Tablas de Bootstrap para listado de productos, inventario y reportes.
- Alertas de Bootstrap (`alert-warning`) para notificar stock mínimo alcanzado.
- Botones primarios para acciones de confirmación (registrar venta, guardar producto).

## Idioma y tono

- Toda la interfaz en español (RNF-03), con lenguaje simple y directo (RNF-01), pensado para personal del supermercado sin formación técnica.
