# navigation-map

> Estado: 🟢 | Última actualización: 2026-07-05
> Autor: Brenda Carolina Galeano Ramírez | Equipo: Desarrollo Supermercado "La Esquina"

```
Login
 └── (según rol)
      ├── Administrador
      │    ├── Productos (listar / crear / editar / eliminar)
      │    ├── Inventario (ver stock, alertas de stock mínimo)
      │    ├── Usuarios (listar / crear / editar roles)
      │    ├── Reportes (ventas por rango de fecha)
      │    └── Ventas (igual que Cajero)
      └── Cajero
           └── Ventas
                ├── Buscar producto
                ├── Agregar al detalle de venta
                ├── Confirmar venta
                └── Ver/imprimir comprobante
```
