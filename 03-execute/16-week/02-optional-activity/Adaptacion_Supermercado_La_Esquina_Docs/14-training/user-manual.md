## Manual del Cajero

1. **Iniciar sesión** con usuario y contraseña asignados.
2. **Registrar una venta**:
   - Buscar el producto por nombre.
   - Indicar la cantidad.
   - Repetir para todos los productos del cliente.
   - El sistema calcula el total automáticamente.
   - Confirmar la venta.
3. **Entregar el comprobante** generado al cliente.
4. Si un producto no tiene stock suficiente, el sistema mostrará un mensaje de error; informar al cliente y avisar al Administrador.

## Consultas del Cliente

- El cliente puede consultar precios directamente en el mostrador con ayuda del cajero.
- Al finalizar la compra, recibe un comprobante con el detalle y el total pagado.
