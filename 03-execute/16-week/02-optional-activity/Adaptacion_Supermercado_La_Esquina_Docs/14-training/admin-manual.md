## Gestión de productos

- Crear, editar o inactivar productos desde el módulo de Productos.
- Definir el stock mínimo de cada producto para recibir alertas oportunas.

## Gestión de inventario

- Revisar el stock actual de todos los productos.
- Atender las alertas de "stock mínimo alcanzado" y reabastecer.

## Gestión de usuarios

- Crear usuarios para nuevos cajeros.
- Asignar el rol correspondiente (Administrador o Cajero).
- Inactivar usuarios que ya no trabajen en el negocio.

## Reportes

- Consultar reportes básicos de ventas por rango de fechas para apoyar decisiones del negocio (por ejemplo, cierre de caja diario).
