# Glosario

> Estado: 🟢 | Última actualización: 2026-07-05
> Autor: Brenda Carolina Galeano Ramírez | Equipo: Desarrollo Supermercado "La Esquina"

| Término | Definición | Contexto |
|---------|------------|----------|
| Producto | Artículo que se vende en el supermercado, con nombre, precio y stock asociado | Dominio |
| Inventario | Registro de existencias disponibles de cada producto | Dominio |
| Venta | Transacción en la que un cajero registra uno o más productos comprados por un cliente | Dominio |
| Detalle de venta | Línea de una venta que asocia un producto, cantidad y subtotal | Dominio |
| Stock mínimo | Cantidad mínima de un producto antes de generar alerta de reabastecimiento | Dominio |
| Cajero | Usuario que registra ventas y consulta productos en el punto de venta | Sistema |
| Administrador | Usuario que gestiona productos, inventario, usuarios y reportes | Sistema |
| Cliente | Persona que compra en el supermercado, consulta precios y recibe comprobante | Sistema |
| Comprobante | Recibo entregado al cliente al finalizar una venta | Dominio |
| Cierre de caja | Proceso de consolidar el total de ventas de un turno o día | Dominio |
| RF | Requerimiento Funcional | SRS |
| RNF | Requerimiento No Funcional | SRS |
