## Dependencias técnicas

| Dependencia | Tipo | Uso |
|-------------|------|-----|
| Python 3.10+ | Runtime | Ejecutar el backend Flask |
| Flask | Framework | Backend web |
| SQLite | Base de datos | Persistencia de datos |
| Bootstrap | Librería CSS | Interfaz del frontend |

## Dependencias externas al proyecto

- Equipo del supermercado (hardware) donde se instalará el sistema.
- Disponibilidad del personal del negocio para pruebas de aceptación y capacitación.
